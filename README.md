2024.6.6
【機械学習】
・データ分析の手順としてまずはEDA(Exploratory data analysis)を行う。
・EDAの手順としては
　①データの確認（カラム数や行数、データの意味など）
　②欠損値の確認（NaNとか0とか""とか）
　③外れ値の確認
　④データのパターンを確認（時系列、季節性、相関など）
